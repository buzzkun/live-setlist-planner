# Live Setlist Planner

React / Next.js / TypeScript の学習用に作成した、ライブのセットリスト管理アプリです。

## 現在できること

- 曲名、雰囲気、演奏時間、メモを入力して曲を追加
- 曲の削除
- 曲数の表示
- 合計演奏時間の表示
- 90分を超えた場合の注意メッセージ表示
- 曲が0件のときの空状態表示

## 学習したこと

- useState
- props
- map
- filter
- reduce
- 条件付き表示
- TypeScript の型定義
- Git / GitHub の基本操作
